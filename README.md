# Smart Attendance Reporter PRO

Desktop application that generates professional Excel attendance reports from ZKTeco attendance devices.

Developed by **Islam Abdeen**

## Features

- Connects directly to ZKTeco attendance devices
- Select date range for attendance extraction
- Filter by employee
- Automatically pair Check-in / Check-out
- Generates formatted Excel reports
- Highlights missing punches
- Calculates working hours automatically

## Technologies

- Python
- CustomTkinter
- Pandas
- OpenPyXL
- ZKTeco SDK
